# Security Policy

## Supported Versions

Only the latest minor versions in the current major line receive security updates.

## Reporting a Vulnerability

Please report vulnerabilities privately to the maintainers. Do not open public issues for undisclosed vulnerabilities.

## Scope

- Connector credential handling
- Redaction and PII leakage
- AuthN/AuthZ bypasses
- Data egress policy violations
- Supply-chain risks
